The following classes provide general functionality: <br><br>
◾ Player : responsible for Pac-Man; <br>
◾ Boundary : necessary for drawing the game barrier; <br>
◾ Ghost : used to create enemies; <br>
◾ Pellet : collectible game objects for Pac-Man; <br>
◾ PowerUp : when collecting an enlarged pellet, turns on the enhanced mode for pacman; <br><br>
The game map supports any transformations at the user's discretion and counts points that using for compiling the top players using local storage.
